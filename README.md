# VoltOrbFlip


## Regeln
Man wird zuerst mit einem 5x5 Spielfeld konfrontiert, indem alle Karten zum Start des Spiels verdeckt sind.
Hinter den einzelnen Felder kann entweder eine 1, eine 2, eine 3 oder ein Voltorb versteckt sein.
Die Ziffern sind die Punktzahl, die sich mit jeder weiteren Ziffer um dessen Wert multipliziert.
z.B. man deckt folgende Karten auf: 1-1-3-2-1.
Dann ist die erreichte Punktzahl 1x1x3x2x1 = 12 Punkte.
Falls ein Voltorb aufgedeckt wird, dann endet das Spiel und man muss neu starten.
Die einzelnen Reihen und Spalten ergeben die am Rand unten und rechts stehenden Felder, die auskunft darüber geben, wie viele Voltorbs in der jeweiligen Reihe/spalte versteckt sind und die zusammenaddierte Punktzahl.
Das Ziel ist es, pro Stufe alle x2 und x3 Felder aufzudecken, um so in die nächste Stufe zu gelangen.
Mit jeder Stufe erhöhen sich die Anzahl and x2 und x3 Felder.
Wenn man 3 mal hintereinander das Spielfeld nicht löst, weil man einen Voltorb aufgedeckt hat, wird man um eine Stufe zurückgesetzt

## Mehrspieler
Man tretet jetzt gegen bis zu 7 andere Spieler an, in einem Wettrennen, wer zuerst eine Punktzahl von 100 erreicht.
Hier kann man die Spielfelder, der anderen, in einer minimalen From sehen und die Fortschrittsbalken der einzelnen Spieler, wer am nähesten an die 100 Punkte ist.
Der Gewinner jeder Runde bekommt auf seine erreichte Punktzahl als Belohnung noch einen x3 Multiplikator.

## Spielgeschäft
Die Gewonnenen Punkte können im Spielgeschäft für Gegenstände eingetauscht werden, die einem kleine Vorteile in den Spielsitzungen geben können, diese garantieren aber nicht den Sieg.
Hierbei ist zu beachten, dass pro Spiel nur ein Gegenstand ausgerüstet werden kann.
Folgende Gegenstände können erworben werden.
- Lupe - Verrät einem zufällig, unter welchem Feld ein Voltorb versteckt ist.
